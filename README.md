# fore-end
前端面试题集合

# 使用说明

1、 拉取代码
`git close https://github.com/zssLove/fore-end.git`

2、 安装依赖
`npm install`

3、 启动服务
`npm run dev`

# 报错
如果出现`error:0308010C:digital envelope routines::unsupported`错误，可能是node版本过高问题，建议使用node 16.x版本，目前我用的16.13.0版本，报错请切换node版本。
